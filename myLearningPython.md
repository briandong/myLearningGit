## variables

支持多变量赋值；python 3.5以后必须严格使用print()格式

```python
a, b, c = 1, 2, 3
print(a, b, c)

=> 1 2 3
```



##while循环

```python
condition = 1
while condition < 4:
	print(condition)
	condition += 1
	
=> 1
2
3
```
